Arrays and Loops

Theory

Arrays and loops are fundamental concepts in JavaScript, offering essential tools for managing data collections and repetitive tasks. This section delves into a comprehensive exploration of both concepts.

Arrays

An array serves as a versatile data structure, enabling the storage of multiple values within a single variable. These values, referred to as elements, are indexed from 0 onwards. Arrays can encompass diverse data types, encompassing numbers, strings, objects, and even other arrays. 


            // Creating an array of numbers
            let numbers = [1, 2, 3, 4, 5];
            
            // Creating an array of strings
            let fruits = ['apple', 'banana', 'orange'];
            
            // Creating an array of mixed data types
            let mixedArray = [10, 'hello', { name: 'John' }];

Elements within an array are accessible using square brackets and their respective indices. For instance: 


            let numbers = [1, 2, 3, 4, 5];
            console.log(numbers[0]); // Output: 1
            console.log(numbers[2]); // Output: 3

Loops

Loops facilitate the repetitive execution of code blocks. They provide the means to iterate over array elements or perform tasks a specific number of times. JavaScript features two primary loop types: the for loop and the while loop.

Example 1: Using a for loop to iterate over an array

            let fruits = ['apple', 'banana', 'orange'];
            for (let i = 0; i < fruits.length; i++) {
                console.log(fruits[i]);
            }

The above loop will output each fruit item within the fruits array to the console.

Example 2: Using a while loop for repetitive tasks

            let count = 0;
            while (count < 5) {
                console.log('Count:', count);
                count++;
            }

The above loop will print the value of count to the console five times, starting from 0 and incrementing by 1 with each iteration.
