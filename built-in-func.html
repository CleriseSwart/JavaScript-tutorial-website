Built-in Functions

Theory

JavaScript provides a rich collection of built-in functions that offer diverse functionalities. These functions are pre-defined and can be employed without explicit declarations. Built-in functions cover various categories, including string manipulation, mathematical operations, date and time handling, and more.

Example 1: String Length


            let text = "Hello, JavaScript!";
            let length = text.length;
            console.log(length); // Output: 18

In this example, the built-in length property of the string object is utilized to determine the character count within the text string.

Example 2: Random Number


            let randomNumber = Math.random();
            console.log(randomNumber); // Output: a random decimal number between 0 and 1

In this case, the built-in Math.random() function is employed to generate a random decimal number ranging from 0 (inclusive) to 1 (exclusive).
