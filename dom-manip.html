DOM Manipulation

Theory

DOM (Document Object Model) manipulation is the process of interacting with the HTML and XML documents using JavaScript. The DOM represents the structure of a document as a tree of objects, where each element of the HTML document is a node in the tree. JavaScript provides several methods to access, modify, and manipulate these DOM elements dynamically.

Example 1: Changing Text Content


            function changeText() {
                // Get the element with the ID 'sample-text'
                let element = document.getElementById('sample-text');
                // Change its text content to 'New Text Content'
                element.textContent = 'New Text Content';
            }

In this example, we create a function changeText that gets an element with the ID sample-text and changes its text content to New Text Content.

Example 2: Adding a New Element


            function addNewElement() {
                // Create a new paragraph element
                let newElement = document.createElement('p');
                // Set its text content
                newElement.textContent = 'This is a new paragraph element.';
                // Append the new element to an element with the ID 'container'
                document.getElementById('container').appendChild(newElement);
            }

In this example, we create a function addNewElement that creates a new paragraph element, sets its text content, and appends it to an element with the ID container.
