jQuery Functions

Theory

jQuery is a fast and lightweight JavaScript library that simplifies many common tasks in web development. It provides a collection of useful functions and methods that make it easy to select HTML elements, manipulate the DOM, handle events, and perform animations. jQuery functions are typically prefixed with the dollar sign ($) and can be used to perform various operations on the selected elements.

Example 1: Hide and Show


            // Hide an element with ID "myElement"
            $('#myElement').hide();

            // Show an element with class "myClass"
            $('.myClass').show();

In this example, we use the jQuery functions hide() and show() to hide and show HTML elements respectively. The # symbol is used to select elements by their ID, and the dot . is used to select elements by their class.

Example 2: Event Handling


            // Click event handler for a button with ID "myButton"
            $('#myButton').click(function () {
                alert('Button clicked!');
            });

In this example, we attach a click event handler to a button with the ID myButton. When the button is clicked, the function inside click() will be executed, displaying an alert with the message "Button clicked!".
