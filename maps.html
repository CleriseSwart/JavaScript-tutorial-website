Maps in JavaScript

Theory

In JavaScript, a Map is a collection of key-value pairs where each key must be unique. Maps are similar to objects, but the main advantage of using Maps is that they allow any data type as a key, whereas objects only allow strings as keys. This enhanced flexibility makes Maps versatile for various use cases, especially when dealing with complex data.

Example 1: Creating a Map


                // Creating a new Map
                let myMap = new Map();

                // Adding key-value pairs to the Map
                myMap.set('name', 'John');
                myMap.set('age', 30);
                myMap.set('isStudent', true);

                // Accessing values using keys
                console.log(myMap.get('name')); // Output: John
                console.log(myMap.get('age')); // Output: 30
                console.log(myMap.get('isStudent')); // Output: true

In this example, we create a new Map named myMap using the new Map() constructor. We then add key-value pairs using the set() method. These key-value pairs allow us to store and retrieve information based on their corresponding keys. The get() method retrieves the value associated with a specific key.

Example 2: Iterating over a Map


                // Creating a new Map
                let fruitColors = new Map();

                // Adding key-value pairs to the Map
                fruitColors.set('apple', 'red');
                fruitColors.set('banana', 'yellow');
                fruitColors.set('grape', 'purple');

                // Iterating over the Map using destructuring
                for (let [fruit, color] of fruitColors) {
                    console.log(`The color of ${fruit} is ${color}`);
                }

In this example, we create a new Map named fruitColors and add key-value pairs representing the color of fruits. We use destructuring to iterate over the Map using a for...of loop. During each iteration, the fruit and color values are extracted and used to compose a descriptive output. This provides an efficient way to process and display the information stored in the Map.
