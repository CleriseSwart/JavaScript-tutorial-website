jQuery Selectors

Theory

jQuery selectors are powerful tools that allow you to select and manipulate HTML elements in a web page. They are used with the $ sign in jQuery. Selectors are similar to CSS selectors and can target elements based on their tag names, classes, IDs, attributes, and more. With jQuery selectors, you can easily access and modify elements to create dynamic and interactive web pages.

Example 1: Select by Tag Name 


                // Select all 
 elements and set their text color to red
                $('p').css('color', 'red');

In this example, we use the jQuery selector $('p') to select all <p> elements in the document and set their text color to red using the css() method.

Example 2: Select by Class 


                // Select all elements with class "highlight" and hide them
                $('.highlight').hide();

Here, we use the jQuery selector $('.highlight') to select all elements with the class "highlight" and hide them using the hide() method.
