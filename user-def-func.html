User-defined Functions

Theory

User-defined functions are custom functions created by the JavaScript programmer to perform specific tasks. These functions allow you to encapsulate a block of code, give it a name, and then call that code by its name whenever needed. User-defined functions are reusable, modular, and can accept parameters and return values.

Example 1: Addition Function


            // Define a user-defined function to add two numbers
            function add(a, b) {
                return a + b;
            }
            
            // Call the function and store the result
            let result = add(5, 3);
            
            // Output the result
            console.log(result); // Output: 8

In this example, we define a user-defined function add that takes two parameters a and b and returns their sum.

Example 2: Calculate Area


            // Define a user-defined function to calculate the area of a circle
            function calculateArea(radius) {
                return Math.PI * radius * radius;
            }
            
            // Call the function and store the area
            let area = calculateArea(5);
            
            // Output the area
            console.log(area); // Output: Approximately 78.54

Here, we create a user-defined function calculateArea that takes the radius of a circle as a parameter and returns the area of the circle.
