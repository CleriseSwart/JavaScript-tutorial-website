Variables

Theory

Variables in JavaScript are containers that store data. They allow us to manipulate and access that data throughout our code. Variables act as named placeholders for values. We can declare a variable using the let, var, or const keywords. Variables can hold different types of data such as numbers, strings, booleans, arrays, objects, and more.

Example 1: Storing a Number


            let age = 25;
            console.log(age); // Output: 25

In this example, we declare a variable named age and assign it the value 25. We can later access and use this value in our code by referencing the variable name.

Example 2: Storing a String


            let message = "Hello, World!";
            console.log(message); // Output: Hello, World!

Here, we declare a variable named message and initialize it with the string value "Hello, World!". We can manipulate or display this string throughout our program.
